## 9. Known Limits

- Not a production-grade sandbox.
- No kernel-level isolation.
- No guarantee against all prompt injection or malicious MCP server behavior.
- No Web Dashboard, RBAC, cloud policy center, or full sandbox.
- Deep scan only calls `initialize` and `tools/list`; it does not execute tools or prove server safety.
- Balanced profile denies non-interactive shell confirmations by default.
